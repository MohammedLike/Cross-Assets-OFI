Why deep learning here?

Cross-Asset Transformer. Takes a sequence of recent OFI values from all assets and uses multi-head self-attention to learn dynamic lead-lag patterns — e.g. which asset's order flow matters at which time, conditional on context.

Graph Attention Network (GNN). Models the universe as a graph where nodes are assets and edges are learned influence weights. Each forward pass passes OFI features along edges and the GAT layer learns which neighbours matter most for predicting the target node's return.

The honest test. If complexity adds value, these models should beat plain Ridge in OOS IC. If they don't, the relationship is approximately linear — consistent with microstructure theory and a publishable finding in itself.
