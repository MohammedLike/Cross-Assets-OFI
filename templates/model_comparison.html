Does complexity add value?

This page is the central honesty check of the project. It puts Ridge (the linear baseline) next to XGBoost, Transformer, GNN, and Ridge with retrieved news features — all evaluated on the same out-of-sample test windows with the same Information Coefficient metric.

Microstructure theory suggests OFI→return relationships are approximately linear at short horizons. If that's right, Ridge should be hard to beat. If a deep model meaningfully wins, that itself is the result. Either outcome is publishable.

Interview Takeaway

For prop / quant interviews

"I tested whether complexity adds value over linear models. Plain Ridge with OFI features captures most of the signal — the Transformer and GNN add a small marginal improvement on some assets but no clear win, which is consistent with microstructure theory that OFI-return relationships are approximately linear at short horizons. The RAG news pipeline adds meaningful context on event-driven days but is noisy on average."
